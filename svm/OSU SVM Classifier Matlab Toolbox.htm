OSU SVM Classifier Matlab Toolbox (ver 3.00)

By Junshui Ma, Yi Zhao, and Stanley Ahalt

What's new in version 3.00?

(1) The core part of this toolbox is now based on LIBSVM version 2.33
(2) It can be run on Windows, Linux (x86), Solaris, and HP-XP.
(3) Two data-preprocessing functions are added, Normalize.m and Scale.m.


1. Why we are here?

Although there are quite a few versions of SVM codes in different languages, including C, Fortran, Java, and Matlab, available on the web, we still found it is necessary to develop a new Matlab SVM toolbox. Matlab is one of the most popular research languages in the world. However, currently, the only Matlab SVM toolbox available is too elementary to deal with any practical problem, which is due to both its high memory requirement and slow computation speed.

Compared with other available SVM packages, our OSU SVM toolbox has the following major features:

(1) SVM Classifier implemented as Matlab toolbox enables its users take full advantage of the convenience of the Matlab platform.

(2) By virtue of Matlab's mex mechanism, this toolbox implements SVM classifiers in C++ using Chih-Chung Chang and Chih-Jen Lin's LIBSVM algorithm (COPYRIGHT_libsvm). Consequently, it is so fast that it is comparable to all the available C-coded SVM packages in speed. Meanwhile, it inherits the memory-management mechanism of LIBSVM, and is capable of dealing with practical classification problem with huge training set.

(3) It is self-contained. That is, it does not request any extra optimization software packages to support it.

(4) It implements not only the fundamental SVM classification algorithm, but also two modified formulations, which are named n-SVM  and one-class SVM classifiers.

3. Install and Learn

After downloading the zipped file, you need to

(1) Create a subdirectory in your Matlab's toolbox directory. (For example: \Matlab\toolbox\osu_svm\)

(2) Unzip the downloaded file into this subdirectory

(3) Add this subdirectory into the search path of your Matlab by "addpath" command, or by the "file->set path.." menu item, including the subdirectory of Demo.

(4) Run the command of "demos" from the Matlab command line to learn how to use this toolbox.
