安卓软件众多，质量参差不齐，每个人也有自己的使用习惯，我就推荐几个自己常用的感觉还不错的软件，抛砖引玉。

[::艾泽拉斯国家地理 BBS.NGACN.CC::]

1 鲜果联播

装过zaker，装过flipboard，最后还是感觉鲜果联播更加人性化。首先界面上是图片，看起来美观好看，其次全文基本都可以显示，这点比flipboard好太多。支持离线下载，可以设置时间自动离线，只在wifi离线。

[ http://pic3.178.com/17/175160/month_1212/ba08e613f5efe4838578276e9c48fa72.jpg ]
[ http://pic3.178.com/17/175160/month_1212/b87ae5f1aa0a9aac687be61221f747f5.jpg ]

[ http://pic3.178.com/17/175160/month_1212/b8471281fc208a789913cb8202fe3d9b.jpg ]


2 Endomondo

水区喜欢跑步骑车的不少，上下班骑车的也很多。这个软件就是我骑车时候用的，记录骑行时间、距离和路线，在网上分享。路线还是很准的，用GPS定位。分享功能很强大。并且骑车时候很好用的一点就是可以设置多长时间没有位置变化自动暂停，移动后继续。

[ http://pic3.178.com/17/175160/month_1212/4be9e344543da7b1ea7e1fc9009aef13.jpg ]
[ http://pic3.178.com/17/175160/month_1212/3c9b699d2fea2a0ba9a344d19de84486.jpg ]
[ http://pic3.178.com/17/175160/month_1212/1d6c2bd773b30a98abd2c990d32d5b92.jpg ]


3 爱壁纸HD

经常看到大家求手机壁纸。这个软件收藏的壁纸很多，高清的。更新也很快，几乎每天都会更新很多壁纸，并且有壁纸自动更换功能(虽然我没用过)。喜欢下壁纸的同学必备。

[ http://pic3.178.com/17/175160/month_1212/998ceeb6a7942601ba888b4bfa3a9278.jpg ]
[ http://pic3.178.com/17/175160/month_1212/65b0a20dc9210005a5b27356fa3774e8.jpg ]


4 BeautyfulWidget

先说缺点，没有墨迹皮肤那么骚，虽然皮肤很多比墨迹还多，不过都是中规中矩的。优点就是干净，没有多余的功能。并且墨迹只要在进程里把墨迹关了，桌面时间就不更新了，这个完全没问题，你关闭所有程序，他后台也会更新。1X1，2X1,4X1,4X2,4X4多种桌面插件可选。还带每小时预报，月相和解锁天气动画。
